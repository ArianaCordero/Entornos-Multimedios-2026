## Clonar un Proyecto

Cada proyecto tiene su propio repositorio. Para clonar uno específico, usar el siguiente comando reemplazando el nombre del proyecto:

```bash
git clone https://github.com/ArianaCordero/MolinoUnity.git
```

---

## Abrir en Unity

1. Abrir Unity Hub
2. Seleccionar Add y luego Add project from disk
3. Navegar a la carpeta del proyecto clonado
4. Abrir la escena principal en Assets/Scenes/
